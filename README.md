# To Do List Web App

A simple to-do list web application built using HTML, JavaScript, and Tailwind CSS.

## Features
- Add new tasks
- Delete tasks
- Filter tasks by status (Pending, In Progress, Done)
- Responsive design
- Status color indicator

## Tech Stack
- HTML
- JavaScript (Vanilla JS)
- Tailwind CSS (via CDN)

## Folder Structure
index.html
js/
  script.js
css/
  style.css

## Notes
The `css/style.css` file is intentionally left empty because this project uses Tailwind CSS via CDN for styling.  
The folder is kept to follow the required project structure.
